# 🔐 Sistema de Login com PHP e MySQL

Sistema de autenticação web desenvolvido com PHP, MySQL e Bootstrap 5.
Permite cadastro de usuários, login seguro com sessões e logout.

## 🛠️ Tecnologias utilizadas

- PHP 8+
- MySQL
- Bootstrap 5.3
- XAMPP (ambiente local)

## ✅ Funcionalidades

- Cadastro de usuários
- Login com validação de sessão
- Senha criptografada com `password_hash`
- Proteção contra SQL Injection com Prepared Statements
- Logout com destruição de sessão
- Redirecionamento automático para login se não autenticado
